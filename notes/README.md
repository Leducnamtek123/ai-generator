# Notes README

Use this folder for short-lived project notes inside Obsidian.

Recommended note types:

- `current/` for active task notes
- `decisions/` for dated decisions
- `research/` for exploratory notes
- `architecture/` for stable project maps
- `daily/` for daily logs and quick capture

Starter notes:

- [Dashboard](Dashboard.md)
- [Project Map](Project%20Map.md)
- [Current Work](Current%20Work.md)
- [Decision Log](Decision%20Log.md)
- [Templates](Templates.md)
- [Daily Note Template](Daily%20Note%20Template.md)

If you want, add daily notes or meeting notes here too. Keep them linked to the source files they describe.
