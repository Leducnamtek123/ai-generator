# Second Brain - Project Knowledge Management

Welcome to the **Wata Meeting Transcription** Second Brain. This system uses a hybrid of **PARA** (Projects, Areas, Resources, Archives) and **Johnny Decimal** for organized documentation.

## 📂 Structure Overview

### 00 PARA (Action-Oriented)
- **01 Projects**: Active tasks and features currently being developed.
- **02 Areas**: Ongoing responsibilities (Backend, Frontend, Infrastructure).
- **03 Resources**: General knowledge, documentation, and references.
- **04 Archives**: Completed projects and historical data.

### 🔢 Johnny Decimal (Taxonomy)
- **11 Management**: Meeting notes and high-level tasks.
- **20 Development**: Deep technical notes for specific components.
- **40 Docs**: Official project deliverables and external documentation.

---

## 🛠️ Getting Started
1. Use the **Templates** folder to create new notes.
2. Link your notes using `[[Internal Links]]`.
3. Use **Tags** for status (e.g., `#todo`, `#in-progress`, `#done`).
