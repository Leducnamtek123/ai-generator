# Wata Meeting Transcription

## 🎯 Overview
AI-powered transcription service with integrated credit-based payment system and multi-channel support (Facebook/Messenger).

## 🏗️ Core Stack
- **Backend**: NestJS, TypeORM, PostgreSQL.
- **Frontend**: React, CSS Modules (with @layer), Shadcn UI.
- **AI**: Whisper v3 (Transcription), GPT-4 (Summarization).
- **Payments**: VNPAY, MoMo, ZaloPay, 9Pay.
